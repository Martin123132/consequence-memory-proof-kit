# Access Request Checklist

Before requesting private access, decide what you need.

Request access here:

https://github.com/Martin123132/consequence-memory-proof-kit/issues/new?template=access_request.md

Do not post confidential details, private email addresses, NDA material, or commercially sensitive plans in a public GitHub issue.

## Evaluation Type

- public proof review only
- private technical review
- private wheel evaluation
- private source review
- benchmark review
- commercial pilot
- production commercial licence

## Information To Provide

- name
- organisation
- role
- GitHub username
- public contact route, or note that private contact is required
- intended use
- requested access level
- whether an NDA can be signed before access
- whether any third-party AI platform, model provider, or hosted service will access the work

## Review Outcome

An access request can be:

- accepted
- accepted with NDA
- accepted with limited artifacts only
- deferred for more information
- declined

Private access is not automatic.
